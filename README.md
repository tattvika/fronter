Fronter
================
Front-end development tools package. 

Dependencies:

- material-design-lite
- moment
- vue

Development dependencies:

- gulp
- gulp-browserify 
- gulp-sass
- browser-sync

#### Installation via npm
``` bash
$ npm install
```

#### Running server
``` bash
$ gulp
```
